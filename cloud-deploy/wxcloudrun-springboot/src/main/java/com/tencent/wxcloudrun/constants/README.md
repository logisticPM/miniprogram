# 状态常量说明

## 活动状态 (ActivityStatus)

| 常量名 | 值 | 说明 |
|-------|-----|-----|
| NOT_STARTED | not_started | 活动尚未开始 |
| IN_PROGRESS | in_progress | 活动进行中 |
| ENDED | ended | 活动已结束 |

## 房间状态 (RoomStatus)

| 常量名 | 值 | 说明 |
|-------|-----|-----|
| AVAILABLE | available | 房间待售 |
| GRABBED | grabbed | 房间已被抢占（但未完成后续流程） |
| SOLD | sold | 房间已售出（完成所有流程） |
